# Hostel Booking Website

A modern, responsive React application for hostel booking services, designed with TypeScript and styled with custom CSS.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Component-Based**: Built with reusable React components for maintainability
- **TypeScript**: Type-safe development with TypeScript
- **Accessibility**: Semantic HTML and accessible navigation

## Sections

1. **Header**: Responsive navigation with mobile menu
2. **Hero**: Eye-catching banner with call-to-action
3. **Features**: Why choose us section with service highlights
4. **Featured Rooms**: Showcase of available accommodations
5. **How It Works**: Step-by-step booking process
6. **Newsletter**: Email subscription section
7. **Footer**: Links and social media

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the project directory:
   ```bash
   cd hostel-booking
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Available Scripts

- `npm start`: Runs the app in development mode
- `npm test`: Launches the test runner
- `npm run build`: Builds the app for production
- `npm run eject`: Ejects from Create React App (one-way operation)
